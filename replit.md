# EverydayAI

AI agent platform for businesses. Deploy intelligent agents to WhatsApp and web widgets.

## Structure

```text
everydayai-v2/
├── frontend/     # React/TypeScript dashboard
├── backend/      # FastAPI API + ARQ worker
└── docker-compose.yml  # Local development
```

Currently `frontend/` and `backend/` only contain placeholder `.gitkeep` files — this is an early scaffold with no application code yet.

## User preferences

- The user is building this project manually with a human engineer, step by step. Do not generate or modify any application code unless explicitly asked in a future request.
